## Package /routes/param

## Constructors

| Constructor                        | Description                                                             |
| :--------------------------------- | :---------------------------------------------------------------------- |
| constructor(name, className=null) | Creates a ***Param class*** instance with the specified parameter name. |
----

## Methods

| Method    | Description                                            |
| :-------- | :----------------------------------------------------- |
| name      | The getter method to get ***Param Name***.       |
